# Dwelltea UI Design Specification

## Overview
This document provides a comprehensive breakdown of all UI components, design elements, and specifications for the Dwelltea real estate application.

---

- **Component**: Dwelltea wordmark with house icon

## 12. RESPONSIVE BEHAVIOR

### Mobile Adaptations
- Navigation collapses to hamburger menu
- Feature cards stack vertically (1 column)
- Hero image moves below text or scales down
- Stats display in 2-column or vertical layout
- Comparable properties scroll horizontally or stack

### Tablet Adaptations
- Navigation may condense
- Feature cards in 2 or 3 columns
- Maintain most desktop layouts with adjusted spacing

---

## 13. COMPONENT STATES

### Buttons
- **Default**: Full color, no special effects
- **Hover**: Slight darkening or brightness increase
- **Active/Pressed**: Further darkening
- **Disabled**: Reduced opacity (50-60%), no interaction

### Links
- **Default**: Standard color
- **Hover**: Underline appears or color change
- **Visited**: May use same color (for consistency)
- **Active**: Brief highlight

### Cards
- **Default**: Standard shadow/border
- **Hover**: May lift (increased shadow) or border highlight
- **Selected**: Not applicable in these screens

---

## 14. ACCESSIBILITY NOTES

- Ensure all interactive elements have sufficient size (44×44px minimum)
- Maintain color contrast ratios (WCAG AA: 4.5:1 for text)
- Provide alt text for all images
- Ensure keyboard navigation for all interactive elements
- Use semantic HTML elements
- Provide focus indicators for interactive elements

## SUMMARY

This design system creates a professional, trustworthy real estate platform with:
- **Warm, earthy color palette** (dark green, cream, gold accents)
- **Clean, modern typography** with clear hierarchy
- **Generous spacing** for readability and focus
- **Lifestyle photography** to create emotional connection
- **Clear CTAs** with high-contrast primary buttons
- **Data visualization** for market insights
- **Icon system** for quick visual recognition
- **Responsive design** for all devices

The overall aesthetic is sophisticated yet approachable, data-driven yet human-centered.
